# شركة رؤى بعيدة - Farvisions Company Website

## نظرة عامة

موقع شركة رؤى بعيدة هو موقع ويب فاخر ومتجاوب مصمم لعرض خدمات الشركة المتخصصة في المقاولات الإنشائية والتشطيبات والنجارة والتصميم الداخلي في المملكة العربية السعودية. يتميز الموقع بتصميم فاخر يعكس الأناقة والتميز في مجال البناء والتشييد.

## الميزات الرئيسية

### 🚀 الصفحات والخدمات
- **الصفحة الرئيسية**: عرض شامل للشركة وخدماتها الإنشائية
- **خدمات المقاولات**: خدمات الإنشاءات والتشطيبات والنجارة الشاملة
- **التصميم الداخلي**: خدمات التصميم الداخلي المتميزة
- **إدارة المشاريع**: خدمات إدارة المشاريع الإنشائية
- **من نحن**: معلومات عن الشركة ورؤيتها
- **اتصل بنا**: نموذج التواصل مع الشركة

### 🎨 التصميم والواجهة
- **تصميم فاخر**: ألوان أنيقة تعكس الفخامة والتميز
- **نظام ألوان متطور**: Navy (#0A2E5A) + Gold (#D4AF37) + Ivory (#F8F5F0)
- **تصميم متجاوب**: يعمل على جميع الأجهزة والأحجام
- **واجهة مستخدم بديهية**: سهلة الاستخدام مع تجربة مستخدم متميزة
- **دعم كامل للغة العربية**: مع اتجاه RTL
- **رسوم متحركة متطورة**: تأثيرات بصرية جذابة وانتقالات سلسة
- **إصلاح مشاكل التباين**: نصوص داكنة تلقائياً على الخلفيات البيضاء

### 🔧 التقنيات المستخدمة
- **Next.js 14**: إطار عمل React الحديث
- **TypeScript**: لكتابة كود آمن ومنظم
- **Tailwind CSS**: لتصميم سريع ومتجاوب
- **Lucide React**: أيقونات عالية الجودة
- **Framer Motion**: للرسوم المتحركة (قابل للإضافة)

### 📱 الاستجابة والسرعة
- **تصميم متجاوب**: يعمل على جميع أحجام الشاشات
- **تصميم موبايل أول**: محسن للأجهزة المحمولة

## الخدمات المقدمة

### 1. خدمات المقاولات (Contracting Services)
- **الإنشاءات السكنية**: بناء المنازل والفيلات
- **الإنشاءات التجارية**: المباني التجارية والمكاتب
- **الصيانة والتطوير**: صيانة المباني القائمة
- **ضمان 5 سنوات**: على جميع الأعمال الإنشائية

### 2. التشطيبات والديكورات
- **تشطيبات الجدران**: دهانات، ورق جدران، ألواح جبس
- **تشطيبات الأرضيات**: بلاط، رخام، باركيه
- **تشطيبات الأسقف**: أسقف معلقة، إضاءات مخفية
- **التشطيبات الخارجية**: عوازل، دهانات مقاومة

### 3. النجارة الفاخرة (Fine Woodworking)
- **أنواع الأخشاب**:
  - خشب البلوط (Oak): قوي ومتين، مثالي للأثاث الفاخر
  - خشب الجوز (Walnut): فاخر بلون بني غامق مع عروق جميلة
  - خشب الماهوجني (Mahogany): أحمر فاخر مع عروق مستقيمة
- **تقنيات التشطيب**:
  - التشطيب بالورنيش: حماية شفافة مع لمعان فاخر
  - التشطيب بالزيت: تشطيب طبيعي يحافظ على رطوبة الخشب
  - التشطيب بالشمع: تشطيب تقليدي يعطي ملمس حريري

### 4. التصميم الداخلي
- **تصميم المنازل**: غرف المعيشة، النوم، المطابخ
- **تصميم المكاتب**: مكاتب إدارية، قاعات اجتماعات
- **تنسيق الأثاث**: اختيار وتنسيق الأثاث والديكورات
- **إضاءات مخصصة**: تصميم أنظمة الإضاءة

### 5. إدارة المشاريع
- **تخطيط المشاريع**: دراسات الجدوى، الجداول الزمنية
- **إدارة الفرق**: اختيار العمالة، التدريب، المتابعة
- **متابعة التقدم**: تقارير دورية، مؤشرات الأداء
- **ضمان الجودة**: مراقبة مستمرة، اختبارات دورية

## التثبيت والتشغيل

### المتطلبات
- Node.js 18+ 
- pnpm (موصى به) أو npm

### خطوات التثبيت

1. استنساخ المشروع
```bash
git clone https://github.com/your-username/favisions.git
cd favisions
```

2. تثبيت التبعيات
```bash
pnpm install
```

3. تشغيل المشروع في وضع التطوير
```bash
pnpm dev
```

4. فتح المتصفح على `http://localhost:3000`

### بناء المشروع للإنتاج
```bash
pnpm build
pnpm start
```

## إصلاح مشاكل التباين

### مشكلة التباين
تم حل مشكلة النصوص البيضاء غير المرئية على الخلفيات البيضاء من خلال:

```css
/* إصلاح تلقائي للنصوص على الخلفيات البيضاء */
[style*="background: #fff"], 
[style*="background: white"] {
  color: #2A2A2A !important;
}

/* إصلاح إضافي للفئات الشائعة */
.bg-white,
.bg-white\/[0-9] {
  color: #2A2A2A !important;
}
```
